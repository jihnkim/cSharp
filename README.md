### C# with vscode/.NET sdk 
---
#### Initialize
1. 
- Extension > C# install
---
2.
- install .NET SDK 

- [Download .NET SDK](https://dotnet.microsoft.com/en-us/download)
---
3.
- Open Terminal
winforms, console and more
```bash
cd c:\workspace

mdk helloCsharp

dotnet 

dotnet new console
dotnet new winforms

dotnet run
```
